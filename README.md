# min and max
```css
p {
  min-width: 200px;
   min-height: 200px;

}
```


```
.container {
  width: 50%;
  height: 200px;
  /* anything outside of the margins of this div would be removed *\
  overflow: hidden;
}

.container img {
  max-width: 100%;
    /* height scales with width *\
  height: auto;
  display: block;
}
```
